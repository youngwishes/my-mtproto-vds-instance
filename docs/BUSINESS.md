# Бизнес-цель

Сервис является частью платформы по продаже MTProto-прокси ключей через Telegram-бота.

## Что продаём

Пользователь покупает временную ссылку на MTProto-прокси (`tg://proxy?...`). Ссылка даёт доступ к прокси-серверу на базе [telemt](https://github.com/nickvnlk/telemt) — это позволяет обходить блокировки Telegram.

Стандартный срок действия ссылки — **30 дней**. По истечении срока ключ автоматически удаляется.

## Ограничения

Каждый пользователь может подключаться максимум с **3 уникальных IP-адресов** одновременно. Это ограничение не позволяет массово распространять одну ссылку — по сути, один ключ рассчитан на одного человека и его устройства.

## Сценарии использования

### Покупка ключа

Пользователь оплачивает доступ через Telegram-бота. Бэкенд генерирует секрет, отправляет запрос на этот сервис, получает параметры подключения и формирует `tg://proxy` ссылку, которую бот отправляет пользователю.

### Перевыпуск ссылки

Если ссылка скомпрометирована (например, случайно отправлена в общий чат и по ней подключились посторонние), пользователь может запросить перевыпуск через бота. Старая ссылка моментально перестаёт работать, выдаётся новая.

### Очистка протухших ключей

Внешний сервис ежедневно по расписанию (celery-задача) собирает ключи с истёкшим сроком действия и отправляет запрос на массовое удаление.
